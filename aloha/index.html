Variaveis

Para criar uma variavel, basta escrever:

var nome = valor

exemplo:

var nome_pessoa = " Joao "

Imprimir 1

Para imprimir uma mensagem, basta escrever o codigo:

put " hello world "

Você tambem pode imprimir variaveis:

var nome = " Julia "

put nome

Imprimir 2

Para imprimir uma mensagem com linha unica, basta escrever o codigo:

puts " hello world "

Você tambem pode imprimir variaveis:

var nome = " Julia "

puts nome

Você tambem pode imprimir espaços:

puts space

e nova linhas

puts newline

funções 1

Deseja criar funções? basta escrever esse codigo:

fnc nome {

codigo

} nome

funções 2

Deseja chamar funções? basta escrever esse codigo:

call nome

exemplo:

fnc mostrarvalor {

var n1 = 10

put n1

} mostrarvalor

call mostrarvalor

verificações if

agora vamos aprender a criar ifs.

escreva isso para fazer um if:

if var1 condicao var2 nome {

codigo

} nome

condições: ==, !=, >, <, >=, <=

exemplo:

var n1 = 10

var n2 = 10

if n1 == n2 verificar {

put " n1 é igual a n2! "

} verificar

loops for

Loops for, uma coisa que precisa muito na programação!

vamos aprender loops for agora!

como escreve um for na linguagem aloha?

assim:

for vezes nome {

codigo

} nome

exemplo:

for 5 repetir {

put " Eu repeti 5 vezes "

} repetir

loops while

Deseja criar loops while? escreva esse codigo!

while nome {

codigo

} nome

exemplo:

while meu_loop {

put " Olá mundo! "

} meu_loop

Sistema

mexendo no sistema

vamos mexer no sistema? escreva isso no seu codigo:

sys " echo hello world! "

o'que o codigo acima faz?: ele executa o comando echo que é do proprio sistema operacional que o usuario esta usando. dentro desse echo, ele executa a mensagem Hello world!

executando a partir de variaveis

var cmd = " echo aaaa "

sys cmd

Mudando valores

deseja adicionar ou subtrair numeros de variaveis? aqui esta um exemplo!:

num increment_var > target_var

exemplo:

var inc = 1

var n = 0

num inc > n

o codigo acima irá adicionar 1 para a variavel n resultando em 1 no valor da variavel n

subtraindo valores

var n1 = -1

var n2 = 1

num n1 > n2

nesse codigo acima, ele subtrai 1 para a variavel n2, resultado em 0 na variavel n2

Parando loops while

essa parte é simples, dentro de um loop while (funciona apenas em um loop while), escreva:

stop

e pronto, o seu loop while foi interrompido

Juntando duas variaveis

juntar duas variaveis na alohalang é muito facil, basta usar:

joinvar var1 >> var2 : outputvar

exemplo:

var cmd = " echo "

var msg = " hello from aloha! "

joinvar msg >> cmd : finalcmd

put finalcmd

isso irá imprimir "echo hello from aloha!" no output

exemplo de execução em um sys:

var cmd = " echo "

var msg = " hello from aloha! "

joinvar msg >> cmd : finalcmd

sys finalcmd

output:

hello from aloha!
