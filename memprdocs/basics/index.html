Commands:

you can use '*' to add 5 to memory, use '+' to add 1 to memory, use '-' to remove 1 from memory, use '!' to remove 5 from memory, use '&' to print the char that's in memory pos, use '%' to print the memory pos.

this language have more commands but it's in input section.

more:

you start in memory pos 26 and 'a' is in pos 26 and 'z' in pos 1. the memory ends in memory pos 64. memory is from 1-64.
